Privacidade de Dados no Agronegócio é um sistema especialista gratuito para verificar a conformidade de empresas do agronegócio com a Lei Geral de Proteção de Dados (LGPD).

Utiliza a ferramenta Balanced Scorecard (BSC) para avaliar o desempenho organizacional, adaptada nas dimensões: Processos, Lei/Norma, Tecnologia e Aprendizado.

Após realizar o diagnóstico você recebe recomendações para auxiliar na tomada de decisões.

SE APLICA

Pessoas ou empresas que realizam operações com dados pessoais (em meios digitais ou em papel) coletados em território nacional para oferecer ou fornecer bens ou serviços.

NÃO SE APLICA

Quando o tratamento de dados é realizado exclusivamente para fins jornalísticos, artísticos, acadêmicos, segurança pública, defesa nacional, investigação e repressão de crimes.
