Privacidade de Dados no Agronegócio

Diagnóstico grátis de conformidade à Lei Geral de Proteção de Dados (LGPD)

Um sistema especialista que utiliza uma adaptação da ferramenta Balanced Scorecard (BSC) para avaliar o desempenho organizacional, com um conjunto de questões objetivas nas dimensões: Processos, Lei/Norma, Tecnologia e Aprendizado.

Após realizar o diagnóstico você recebe recomendações para auxiliar na tomada de decisões.

SE APLICA

Pessoas ou empresas que realizam operações com dados pessoais (meios digitais ou papel) coletados em território nacional para oferecer ou fornecer bens ou serviços.

NÃO SE APLICA

Quando o tratamento de dados é realizado exclusivamente para fins jornalísticos, artísticos, acadêmicos, segurança pública, defesa nacional, investigação e repressão de crimes.
